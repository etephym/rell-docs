## Пассивки менторов

1. **Chi Dash** - Активация: С+С. КД: 3 сек. Вы делаете быстрый рывок вперёд, хитбокс которого ломает обычный/улучшенный блок.
   - Требуется 6400 чи и 6400 нин

2. **Instant Rush/Faster STM Regen** - Ваш раш (W+W) теперь мгновенный и в нем можно юзать скиллы, но всё ещё не пробивает улучшенный блок. Стамина восстанавливается значительно быстрее.
   - Требуется 7100 тай

3. **Stronger Block** - Большая часть блокбрейкеров (ББ) не ломают ваш блок.
   - Требуется 7000 хп

4. **Wider Dash** - Ваш деш становится длиннее.
   - Требуется 6200 тай

5. **Faster/Passive Charging** - Чакра восстанавливается в 1.5-2 раза быстрее.
   - Требуется 6500 нин

### Jiso Seishin (R1/C3)

- Стронг блок, лучший танк в игре
- Хорош в командных боях (рейды, вары)

**Хорош с расами:** Celestial, Puppet

### Shindai Akuma (R2/C2) ✮

- Самый мобильный ментор. Chi Dash + Wider Dash
- Сбалансированные статы, подходит для всего
- Лучший в геймплее

**Хорош с расами:** Puppet, Celestial

::: warning
Самый сложный в освоении — требует понимания механик
:::

### Narumaki (R3/C4)

- Больше чакры, больше дамага
- **Только** для командных боев

::: danger Не качай соло
Narumaki бесполезен в 1v1 — только рейды и вары
:::

**Хорош с расой:** Celestial

### Bruce Kenichi (R4/C1)

- Мгновенный раш, спам тайджутсу
- Только для 1v1

**Хорош с расами:** Celestial, Puppet, Snake
