# Проблемы Shindo Life

> Актуально с **~01.03.2025** · by lewisky

---

## 1. Бан игроков через зелёные квесты

Реллы добавили в игру античит против автофарма, который не менялся с релиза.

**Как работало:** при взятии зелёного квеста появляется дополнительный энтити далеко за картой. При телепортации к нему аккаунт сразу банился. Обойти по координатам несложно.

Нашлись люди, которые начали **абьюзить это для банов** случайных игроков — только на картах с зелёными квестами.

**Пример:** один из скиллов Ray Kerada отлично подходит чтобы захватить игрока и телепортировать его к той сущности. Цель фиксируется снизу, пока ты сверху.

**Почему упал онлайн?** Люди банили по ~500 невинных игроков на Эмбере ежедневно. Апелляции по этой категории не принимаются — бан вынесен самой игрой.

::: danger Не находитесь на картах с зелёными квестами
:::

---

## 2. Баны фейковыми клипами

Использование некомпетентности модераторов — бан любого игрока поддельным роликом (например, через смену ника скриптом). Модераторы либо не смотрят жалобы, либо не играют в Roblox.

Половина банов — за внутриигровые механики или редкие баги самой игры:

- У мода Aizen есть механика увеличения дистанции некоторых скиллов (Ashura и т.д.) — модераторы об этом не знают
- Если тебя убивают сквадом и их скилл тебя багает — ты резко улетаешь. Это тоже бан, хотя ты ни в чём не виноват

## 4. Откат данных игрока

Использование уязвимости — установка некорректного значения в дату игрока, что вызывает откат / несохранение:

```lua
game:GetService("Players").LocalPlayer.startevent:FireServer("band", "\128")

local tpsrv = game:GetService("TeleportService")
tpsrv:Teleport(game.PlaceId, game.Players.LocalPlayer)
```

Используют для:
- Выдачи неограниченного количества RC через RELL bloodline сбросом кулдауна
- Лёгкого получения расы Celestial
- Получения высокого ранга через покупку за Rell Coins с последующим откатом

::: info Rank, Pro Rank и Rep сохраняются отдельным путём — откатить невозможно
:::

## 5. Покупка лимиток через замену ID

Получение любых синих лимиток и предметов через замену `[ID]` в Rell Shop на нужный bloodline / element / item.
